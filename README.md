## WakeMeUp is a mobile app that automatically wakes you up when you're approaching your destination — no more sleeping past your stop.

### At A Glance
<img width="735" height="502" alt="Image" src="https://github.com/user-attachments/assets/8043bd1f-1170-43f6-b15d-5bfe9517fcfc" />

### Tech Stack
React Native + Expo   — mobile frontend
<br>
Express + Node        — REST API backend
<br>
PostgreSQL (NeonDB)   — user & alarm & location storage
<br>
Google Geocoding API  — location to coordinates
<br>
JWT                   — authentication

### How to Run 
```
git clone https://github.com/wz-Tan/WakeMeUp.git
npm install
```

Terminal 1:
<br>
```
cd app
npm run start
```

Terminal 2:
```
cd backend
node server.js
```
